# نور القرآن — Noor Quran 🌙

A beautiful, elegant, and fully functional Quran reading website with translations in **23 languages**.

## ✨ Features

- 📖 **All 114 Surahs** — Complete Quran with authentic Arabic Uthmani text
- 🌍 **23 Languages** — English, Urdu, Hindi, Korean, Japanese, Chinese, German, Italian, French, Spanish, Turkish, Indonesian, Bengali, Russian, Persian, Malay, Dutch, Portuguese, Swahili + more
- 🎨 **Beautiful Design** — Dark elegant theme with gold accents, proper Arabic Naskh typography
- 🔍 **Search** — Instantly search any Surah by name, number, or meaning
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet, and desktop
- ⌨️ **Keyboard Shortcuts** — Arrow keys for navigation, `/` to focus search
- 🔖 **Bookmarks** — Save your favorite ayahs (stored in localStorage)
- 📋 **Copy Ayahs** — Copy Arabic + translation with one click
- 🔤 **Font Size Control** — Small / Medium / Large
- 📊 **Reading Progress** — Visual progress bar as you scroll
- ⚡ **Fast** — No build step, pure HTML/CSS/JS, CDN-hosted fonts

## 🚀 Usage

Simply open `index.html` in any browser — no server required, no build step.

## 🔌 API

Uses the free [AlQuran.cloud API](https://alquran.cloud) — no API key required.

| Key | Action |
|-----|--------|
